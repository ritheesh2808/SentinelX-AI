# SentinelX AI

Enterprise-ready AI security platform starter with separated frontend and backend workspaces.

## Technology Stack

- React
- TypeScript
- Vite
- Tailwind CSS v4
- Express
- Node.js

## Folder Structure

```text
SentinelX-AI/
├── backend/   # Express API server
├── docs/      # Project documentation
├── frontend/  # React Vite application
└── README.md  # Project overview
```

## Current Status

Project stabilization is in progress. The frontend and backend are configured for local development, with database, authentication, Prisma, and PostgreSQL implementation intentionally deferred.
